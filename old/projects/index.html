Pace Spotter (https://pacespotter.kylebaldw.in/)

When I started running I tried all of the running playlists and was frustrated keeping track of what BPM equaled which pace and listening to the same music all the time is no fun. So I created an app that allows you to put in a seed track (for style) and takes either BPM or pace information and builds a playlist for you and adds it to your Spotify account.

Built with Angular 2+ and uses the Spotify API
